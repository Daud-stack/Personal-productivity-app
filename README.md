# 🚀 Productivity Command Center

A high-performance, premium personal productivity application designed for deep work, habit mastery, and goal visualization. Built with a modern **Bento Grid** layout and featuring a behavioral-science-backed "Deep Work Focus Hub."

## ✨ Key Features

### 🍱 Bento Grid Dashboard
A sleek, modern layout inspired by world-class design standards (Apple, Linear, Stripe). 
* **Hierarchical View**: Your most critical data—Habits and Performance—is front and center.
* **Glassmorphism**: A premium UI with subtle blurs, gradients, and soft shadows.

### ⏱️ Deep Work Focus Hub
Integrated Pomodoro-style timer to battle distractions.
* **SVG Interactive Timer**: Beautifully animated circular timer.
* **Session Logging**: Log your deep work minutes directly into your performance stats.
* **Goal Tracking**: See how close you are to your daily focus target.

### 🌓 Performance Dark Mode
Fully optimized for late-night productivity.
* **Midnight Slate Theme**: Reduced eye strain while maintaining high contrast.
* **Smooth Transitions**: 0.4s seamless fade between themes.
* **Theme-Aware Charts**: Data visualizations automatically adapt their colors.

### 📈 Intelligent Analytics
* **Momentum Score**: Tracks improved performance vs. the previous week.
* **Weekly Retrospective**: Score your progress across Health, Career, Finance, and Personal goals.
* **Micro-Learning**: Daily productivity tips with a personal "Save to Library" feature.

## 🛠️ Technology Stack

* **Backend**: Python / Flask
* **Database**: SQLite (SQLAlchemy)
* **Frontend**: Vanilla JS (ES6+), Bootstrap 5 (Modified), Chart.js, AOS (Animate on Scroll)
* **Design**: Custom CSS Grid with Glassmorphism / Dark Mode Support

## 🚀 Quick Start

### 1. Clone the repository
```bash
git clone https://github.com/Daud-stack/Personal-productivity-app.git
cd Personal-productivity-app
```

### 2. Set up virtual environment
```bash
python -m venv venv
# Windows:
.\venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Initialize Database
```bash
python init_db.py
```

### 5. Run the Engine
```bash
python run.py
```
Open your browser to `http://127.0.0.1:5000`
